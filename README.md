# Crispin Beatz - Music Producer Portfolio

A modern, high-end portfolio website for music producer Crispin Beatz, built with React and TailwindCSS.

## Features

- Responsive design with mobile-first approach
- Dark theme with neon accents and glassmorphism effects
- Animated UI elements using Framer Motion
- Interactive beat player section
- Services showcase
- Portfolio gallery
- Testimonials section
- Contact form with social media integration

## Tech Stack

- React 19
- TailwindCSS
- Framer Motion (for animations)
- React Icons
- Vite (build tool)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Customization

- Colors can be modified in `tailwind.config.js`
- Fonts are loaded from Google Fonts (Inter and Playfair Display)
- All content is easily editable in the component files

## Deployment

This project can be deployed to any static hosting service like Vercel, Netlify, or GitHub Pages.
